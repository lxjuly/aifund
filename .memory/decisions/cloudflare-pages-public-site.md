# Publish The Public Site On Cloudflare Pages

AIfund's public website is hosted on Cloudflare Pages.

This is preferred over Vercel. The initial Astro scaffold, analyze endpoint, and
Cloudflare Pages deployment notes are in place under `apps/web`.

Public pages avoid regulated fund-offering language and include a disclosures
page.
