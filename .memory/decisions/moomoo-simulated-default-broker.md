# Moomoo OpenAPI Simulated Is The Default Broker

AIfund's default broker target for new execution work is Moomoo OpenAPI
simulated trading.

This supersedes Alpaca as the default. The Alpaca adapter is retained only as an
optional or historical path.

Simulated trading remains the default execution path. This decision does not
enable live trading.
